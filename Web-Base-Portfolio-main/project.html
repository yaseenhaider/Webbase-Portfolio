Project 3: Cipher Text

A command-line program that encrypts messages using Caesar’s cipher. Features: Accepts a single non-negative integer as a command-line argument. Validates input: Ensures exactly one argument and that it contains only digits. Encrypts user-inputted plaintext by shifting each alphabetical character k positions forward, wrapping around if needed. Leaves non-alphabetical characters unchanged. Outputs the ciphertext followed by a newline and exits cleanly.
